Show a list of entries related to the present post or page with a brief introduction.

== Description ==

Show a list of entries related to the present post or page with a brief introduction. Using text format to show the list it is posible to engaged your readers in a more effective way. The plugin shows a list of 8 entries related to the actual post, with a brief introducttion of 43 word taked from the begging of the entry o page.

Tested to reduce de bounce rated and improve the permanence of the readers.


== Installation ==

1. Upload `social-fellow` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
